# 2026-05-30

## 修复导出Excel弹窗 - 浏览按钮报错问题
- **问题**：导出Excel弹窗中点击"浏览..."按钮后，文件夹浏览器无法正确显示内容，路径无效时直接报错
- **修复**：
  1. `server.js` - `/api/list-dirs` 接口增加路径存在性检查，不存在时回退到 `os.homedir()`
  2. `HJ.html` - `loadFolderList` 函数增加 `isRetry` 参数，加载失败时自动回退到根目录（显示所有盘符）

## 优化导出Excel弹窗 - 去掉自定义文件夹浏览器
- **问题**：用户反馈自定义文件夹浏览器弹窗操作麻烦，希望简化路径选择
- **修改**：
  1. 完全删除 `folderBrowserDialog` 自定义弹窗及其所有相关 JS 函数（`browseExportFolder`, `openFolderBrowser`, `closeFolderBrowser`, `confirmFolderBrowser`, `loadFolderList`, `renderFolderList`, `loadParentFolder`）
  2. 删除"浏览..."按钮
  3. 保留"桌面"和"自定义路径"两个选项
  4. "自定义路径"模式下路径输入框可编辑，用户直接输入或粘贴路径
  5. 添加常用路径快捷按钮（桌面、文档、下载），选择"自定义路径"后显示，一键填入常用路径

## 导出文件命名规则优化
- **需求**：两处导出 Excel 的文件名改为 `YYYYMMDD 头文件+用户输入名称.xlsx`，Excel 内部标题及工作表名不变
- **修改**：
  1. `HJ.html`：
     - `exportMode='selected'` 默认名改为"原材料清单明细表"，`exportMode='price'` 默认名改为"外购仪器设备"
     - 新增 `getDateStr()` 生成 YYYYMMDD 日期字符串
     - 新增 `updateExportDesc()` 动态更新保存文件名提示
     - `doExportToFile` 构建 `diskFileName`：selected 模式固定 `日期 原材料清单明细表`，price 模式 `日期 外购仪器设备——用户输入`
     - 弹窗内描述文字（`export-desc`）实时显示最终文件名
  2. `server.js`（两处 `/api/export-to-file` handler）：
     - 接收新字段 `diskFileName`，用于磁盘文件路径
     - `fileName` 保持用于 Excel 内部标题（`safeTitle`）和工作表名
     - 文件名去重逻辑改用 `safeDiskName`
