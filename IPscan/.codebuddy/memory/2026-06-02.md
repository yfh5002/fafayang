## import re.py - 外网检测网络配置
添加外网检测的网络配置规格展示：

- 新增 `on_ip_input_change()` 自动更新网关
- 格式：IP地址/子网掩码/网关/DNS/备用DNS 五行

## import re.py - 外网检测机制重构（三层递进式）
针对具备外网权限的IP被误判为断网的问题，重构了 `check_ip_has_internet` 检测机制：
- 第一层 TCP连接：预解析IP直连（百度/Cloudflare/Google/阿里），每个目标失败重试2次
- 第二层 HTTP验证：发起GET请求验证HTTP状态码(200/301/302/403/404均认为可达)
- 第三层 DNS查询：UDP 53端口向DNS服务器发送A记录查询
- 评分机制：TCP 60分 + HTTP 25分 + DNS 15分，≥60分判定有外网
- 目标池全部使用预解析IP地址，避免DNS依赖
- 移除对连接拒绝错误的排除，不再将ConnectionRefusedError当作失败
- 隐藏控制台窗口
