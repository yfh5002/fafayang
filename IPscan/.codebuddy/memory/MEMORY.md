# IPscan 项目记忆

## 项目概述
- 名称：内网设备扫描检测工具
- 文件：`import re.py`（Tkinter GUI 单文件）
- 功能：扫描内网在线设备，检测哪些设备具备外网访问能力
- 运行要求：管理员权限（自动提权）

## 技术要点
- TCP 外网探测目标池已扩展至 11 个多运营商 IP（百度电信/联通、Cloudflare、Google、阿里、腾讯、GitHub）
- HTTP 验证目标 4 个，DNS 服务器 6 个（国内外冗余）
- 判定逻辑：本机自检 → 网卡级测试 → 网关检测 → 自检兜底
- 自检第一轮并行 TCP 探测，全失败后串联重试（避免瞬时网络波动误判）
- 网卡级测试先绑包 TCP，失败回退纯 TCP，再失败用自检结果兜底
- check_ip_has_internet 三层判定：网卡匹配 → 网关检测 → 自检补正
- 扫描范围 .1 ~ .254（跳过 .0 网络地址和 .255 广播）
- 输出报告到 `C:\Users\Administrator\Desktop\内网设备报告.txt`
