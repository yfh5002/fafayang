SSL 证书安装向导

为什么需要安装证书？

HJSYSTEM 使用 HTTPS 加密连接保护您的数据安全。由于使用的是自签名证书，浏览器会显示安全警告。

⚠ 未安装证书时，导出 Excel 文件将下载到浏览器默认"下载"文件夹，无法选择保存到桌面。

安装证书后，警告消失，导出时可自主选择保存位置（如桌面）。

步骤 1：下载证书

点击下方按钮下载证书文件（如果按钮无效，也可以直接访问 /ca.crt）：

备用下载 /ca.crt

下载后双击安装证书，或右键「安装证书」

步骤 2：安装证书

  1. 双击下载的 ca.crt 文件
  2. 点击"安装证书"按钮
  3. 选择"本地计算机"，点击"下一步"
  4. 选择"将所有的证书都放入下列存储"，点击"浏览"
  5. 选择"受信任的根证书颁发机构"，点击"确定"
  6. 点击"下一步" → "完成"
  7. 如果弹出安全警告，点击"是"确认

步骤 3：重启浏览器

重要！ 必须完全关闭所有浏览器窗口（包括其他标签页），然后重新打开浏览器。

建议直接重启电脑，确保证书完全生效。

步骤 4：访问系统

证书安装完成后，访问以下地址：

https://服务器IP:5002/HJ.html

安全警告应该已经消失。

立即访问系统

仍然显示不安全？

如果安装证书后仍然显示不安全，请按以下顺序排查：

  • 1. 确认安装的是最新证书：服务器重新生成证书后，必须重新下载安装。旧证书无效。
  • 2. 确认安装位置正确：必须安装到"受信任的根证书颁发机构"，不是"个人"或"中级证书颁发机构"
  • 3. 重启电脑：完全重启系统，不要只重启浏览器
  • 4. 检查系统时间：确保电脑系统时间正确
  • 5. 360/安全软件：某些安全软件会拦截自签名证书，尝试暂时关闭或添加信任
  • 6. 重新生成证书：如果以上都无效，在服务器端运行 python generate-cert-python.py 重新生成，客户端重新下载安装

Edge/Chrome 浏览器特殊说明

新版 Edge/Chrome 对自签名证书有更严格的要求。如果安装后仍显示不安全：

1. 在地址栏输入 chrome://flags/#allow-insecure-localhost（Edge 同理）

2. 找到 "Allow invalid certificates for resources loaded from localhost"

3. 将其设置为 Enabled

4. 重启浏览器

提示

• 证书只需安装一次，之后访问系统不会再显示安全警告

• 如果有多台电脑需要访问，每台电脑都需要安装证书

• 证书有效期为 1 年，过期后需要重新安装
